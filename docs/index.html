Self-hosted

Running in 5 steps

Deploy to Railway with your own Monobank token. Full control, no subscriptions.

01
Clone the repo
git clone https://github.com/nkhimin/hermes-agent
cd hermes-agent
02
Set environment variables
FINANCE_BOT_TOKEN=your_telegram_bot_token
MONO_TOKEN=your_monobank_token
GROUP_CHAT_ID=your_telegram_group_id
MINI_APP_URL=your_mini_app_url
03
Deploy to Railway
railway up --detach
04
Add bot to Telegram group
Add @sova_finance_bot to your Telegram forum group as admin with "Post messages" permission.
05
Configure topics
Run /setup in the Finance topic. The bot auto-discovers thread IDs and saves them to config.

Built to be extended.

One base class. One config file. One line to add a new agent.

main.py 
# register agents here
agents = [
  FinanceAgent(config.finance),
  # NewsAgent(config.news),
  # CustomAgent(config.x),
]

# one line to add a new agent
await asyncio.gather(
  *[a.run_polling() for a in agents],
  queue_sender.run(),
)
